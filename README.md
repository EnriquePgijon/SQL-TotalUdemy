## Sobre el curso

**SQL Total** es un curso completo de SQL en Udemy que abarca desde los fundamentos hasta conceptos avanzados como procedimientos almacenados, transacciones y optimización de consultas, con una duración de **12 días** de contenido.

---

## Tecnologías y herramientas

- **Base de datos:** MySQL 8.x
- **IDE:** MySQL Workbench
- **Plataforma:** Udemy
- **Control de versiones:** Git & GitHub

---

## Objetivo

Completar el curso de 12 días adquiriendo una base sólida en SQL, desde consultas básicas hasta administración de bases de datos relacionales y proyectos prácticos reales.
